# vue-crud-json-server

Proyecto base creado con Vue 3 y Vite para una futura aplicacion CRUD.

## Tecnologias iniciales

- Vue 3
- Vite
- JavaScript
- npm

## Instalacion

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## API fake

Para levantar json-server:

```bash
npm run server
```

La API queda disponible en:

```text
http://localhost:8080/posts
```
